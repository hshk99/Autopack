# Archived Configuration Files

This directory contains configuration files that were created as planning/design documents
but are not actively used by the codebase.

## Files

### feature_catalog.yaml
Feature catalog for reusing code from known repositories. Part of v7 architect recommendations
for feature reuse. Not currently integrated with any codebase logic.

### stack_profiles.yaml
Stack profile configuration defining technology stacks for different project types.
Part of v7 architect recommendations. Not currently integrated with any codebase logic.

### tools.yaml
Tool categorization configuration. Some referenced scripts exist, others do not.
Configuration file is not actively used by any code.

## Status

These files are preserved for reference and potential future implementation.
They were archived as part of gap analysis item 2.11 (Config surface area drift).
